
# IndieTimer

**A beautiful, privacy-first time tracker for freelancers, indie devs, and solopreneurs.**

Track hours, stay focused with Pomodoro, and manage your goals — all in a single offline HTML file that runs directly in your browser.

[**⬇️ Download IndieTime**](indietime.html)

---

## ✨ Features

- **Live Timer** – One-click clock in / clock out
- **Manual Entries** – Easily add or edit past sessions
- **Goal Progress** – Daily, weekly, and monthly goals with visual bars
- **Built-in Pomodoro** – Focus timer with break mode
- **Activity Log** – Search, filter, sort, and review all entries
- **Recent Tasks** – Quick selection of frequently used tasks
- **CSV Export & Import** – Full backup and restore
- **Dark Mode** – Automatic + manual theme toggle
- **Fully Offline** – Everything saved in your browser (no accounts)
- **Responsive** – Works well on desktop and mobile

## How to Use

1. Download `indietimer.html`
2. Open the file in any modern browser (Chrome, Edge, Firefox, Safari)
3. Start tracking your time immediately

**Tip:** Bookmark the file or pin it to your dock/taskbar for fast access.

## Data & Backups

## Data & Backups (Important)

IndieTimer automatically saves all your data in your browser using **LocalStorage**. This means: - Your entries, settings, and progress are saved automatically. - You don’t need to manually save after every entry.

**⚠️ However, browser-stored data can be lost if:** 
- You clear your browser cache or history 
- You use Incognito / Private browsing mode 
- The browser storage gets corrupted (rare) 
- You switch to a different browser or device

### How to Keep Your Data Safe

1. **Regular Backups** – Click **"Export Backup"** in the app (top controls). This downloads a `.csv` file with all your entries. 
2. **Restore Data** – Use the **"Import"** button to load a previously exported backup. 
3. **Best Practice**: Export a backup at the end of each week.
